트위터 기능

Twitter회사에서 사용하는 글자수140자 제한하는 핵심 기능을 구현해보았습니다.

1. 테스트를 위해 Lorem ipsum문구를 프롬트창에 입력해보겠습니다.

2. 글자수가 제한되지 않았을때 전체 글자수를 alert창에 띄운모습입니다.

3. limitTweet 변수에 slice()기능을 적용하여 140자로 제한된 문구가 alert창에 출력된 모습입니다.

4. 이 모든 과정을 한줄에 요약하였을때도 위와 같은 결과물을 출력하는 연습을 해보았습니다.
